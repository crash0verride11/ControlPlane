About Evidence Sources

Evidence Sources are ways ControlPlane can determine where you are or what you are doing. You enable Evidence Sources so that you can then create rules that define when to transition to a configured context.

ControlPlane current supports the following evidence sources:

  • Active Application: Monitors which application has focus
  • Attached Power Adapter: Detects which power adapter is attached to your Mac based on its serial number. Useful on portable Macs only
  • AudioOutput: Monitors which audio output device is in use. Could be headphones, internal speakers or whatever else is available on the system
  • Bluetooth: Connected and/or nearby bluetooth devices
  • Bonjour: Services and devices located on your local network
  • CoreLocation: Your current location
  • FireWire: Connected FireWire devices
  • IP: Current assigned IP addresses (IPV4 only)
  • Light: How bright is your location, provided by the ambient light sensor
  • Monitor: Attached monitors
  • NetworkLink: Status of a network link, either up or down
  • Power: Monitors the source of power, battery or mains
  • RunningApplication: Monitor running applications
  • ShellScript: Allows you to write your own evidence source based on any script
  • Sleep/Wake: Allows ControlPlane to react to sleep and wake events
  • TimeOfDay: Match a given time
  • USB: Monitor attached USB devices
  • WiFi: Connected or nearby WiFi access points
